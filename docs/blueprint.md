# **App Name**: Nebula Finanx

## Core Features:

- Dashboard Summary: Display a comprehensive overview including the overall balance, total income, and total expenses in a clean, card-based layout.
- Transaction Listing: Show a chronological list of all recent income and expense transactions with key details like amount, date, and category.
- Transaction Entry Form: Provide a form to easily add new income or expense entries, specifying amount, date, and a brief description.
- Financial Flow Chart: Visualize the monthly financial flow (income vs. expenses) through an interactive chart, mimicking the repository's design.
- Basic Navigation: Implement a simple navigation structure, as indicated by the bottom bar, to switch between primary views (e.g., Dashboard, Transactions, Advisor).
- AI Financial Insight Tool: A tool that provides generic financial tips or highlights simple spending patterns based on current data, accessible via a dedicated 'Advisor' section.

## Style Guidelines:

- Color Scheme: Dark mode to reflect a sophisticated, futuristic 'Nebula' theme.
- Background Color: A very dark desaturated indigo (#1B1824) to create a deep, immersive environment.
- Primary Color: A dark blue-violet (#59497A) for cards, sections, and primary interactive elements, contrasting with the background.
- Accent Color: A bright, electric blue-green (#4DD0E1) for highlights, active states, and calls to action, maintaining a futuristic feel.
- Income Indicator: A vibrant green (#42D05C) for positive financial values.
- Expense Indicator: A subtle red (#993B3B) for negative financial values.
- Headline font: 'Space Grotesk' (sans-serif) for titles and numbers, conveying a modern, techy aesthetic. Body font: 'Inter' (sans-serif) for all other text, ensuring readability and a neutral feel.
- Utilize simple, geometric line icons with a subtle futuristic flair, consistent with the repository's original design, for navigation and feature representations.
- Dashboard layout should feature distinct 'cards' or modules for key data summaries, with a prominent chart section. Employ clear visual hierarchy with ample dark space and organized content blocks.
- Subtle, smooth transitions for UI element states (e.g., card hovers, button presses) and data updates (e.g., chart loading) to enhance the modern, premium feel without distraction.
